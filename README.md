## 🚀 Instalação

**Clone o Repositório**

```
git clone https://github.com/rodrigosuelli/semana-omnistack-11.git

cd semana-omnistack-11
```

**Instale as dependências com NPM ou Yarn**

```
npm install
//ou
yarn
```
